SectionedRecyclerViewAdapter

abstract class SectionedRecyclerViewAdapter<H : ViewHolder, VH : ViewHolder> : Adapter<ViewHolder>

An extension to RecyclerView.Adapter to provide sections with headers to a RecyclerView. Each section can have an arbitrary number of items.

Modified version of SectionedRecyclerView adapter without the footers and converted to Kotlin.

Parameters

H - Class extending RecyclerView.ViewHolder to hold and bind the header view

VH - Class extending RecyclerView.ViewHolder to hold and bind the items view
