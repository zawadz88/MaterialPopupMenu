MaterialRecyclerViewPopupWindow

class MaterialRecyclerViewPopupWindow

A more Material version of ListPopupWindow based on RecyclerView.

Its width is a multiple of 56dp units with a minimum of 112dp and a maximum of 280dp as stated in the Material documentation
